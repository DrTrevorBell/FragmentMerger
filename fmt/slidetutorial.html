Fragment Merger Tool


Slide Tutorial When merging fragments which constitute a full-length circular genome (such as a viral genome), it is likely that the final merged sequence will be in the correct orientation. For example, the hepatitis B virus (HBV) genome is numbered starting from 1 according to the position of the EcoR1 restriction site. Full length HBV sequences created with the tool will not start from this co-ordinate, as the sequence data will run from the position of the sequencing primer which is specified as the first fragment.

If values are provided for the optional pair of parameters named "Slide Motif" and "Slide Position", the tool will attempt to slide the final merged sequence such that the slide motif specified is places at the sequence position specified. If fragments constituting a full-length HBV sequence are provided, the EcoR1 motif could be specified for the "slide motif" with a "slide position" value of 1. Alternately, another highly-conserved motif could be specified, such as the "ATAGGA" motif found at position 1750.

Although slide paremeters can be specified for non-full-length merges (and the tool will slide the sequence accordingly if possible), this typically would not be appropriate, as sliding linear fragments does not make sense biologically.
